    Introduction

    reco_luan 4/9/2019

    Welcome to vuepress-theme-reco@1.x! If you are using vuepress@0.x, please move to vuepress-theme-reco@0.x.

    This theme almost inherits all the features of the VuePress default theme, so this document only serves to introduce extra features. If you find that some features of this site do not have related documentation, or if you want to know some features of the default theme, please go to the official documentation (opens new window).
